# **App Name**: DriveVault

## Core Features:

- Connect to Google Drive: Allows users to securely connect their Google Drive account to the application.
- Initiate Manual Backup: Enables users to manually start a backup of their data from the connected service to Google Drive.
- View Backup Log: Presents a log of all backup activities, including timestamps, status, and any relevant messages.

## Style Guidelines:

- Primary color: Deep blue (#2962FF) for trust and security.
- Background color: Light gray (#F0F4F8), creating a clean and professional look.
- Accent color: Soft blue (#5ECEF1) to highlight key interactive elements and calls to action. 
- Body and headline font: 'Inter', a grotesque-style sans-serif.
- Use modern, minimalist icons to represent different data types and backup statuses.
- Employ a clean and organized layout, ensuring easy navigation and readability.
- Incorporate subtle animations and transitions to provide feedback and enhance user experience during backup processes.
